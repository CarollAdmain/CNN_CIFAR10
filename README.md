# CNN_CIFAR10

本Demo是tensorflow利用卷积神经网络实现对CIFAR10进行图像分类，load_data.py用于加载数据，model.py定义了CNN模型，包括 conv1、pool1 && norm1、conv2、pool2 && norm2、full-connect1、full_connect2、softmax。
train.py是用于将模型在CIFAR-10数据集上训练，test.py用于测试模型分类效果。
